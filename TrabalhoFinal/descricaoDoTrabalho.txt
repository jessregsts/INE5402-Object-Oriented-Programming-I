2) Descrição do Trabalho - Aplicação
Neste trabalho, você deverá pesquisar e estudar orientação a objetos na linguagem Python e desenvolver uma aplicação qualquer, que possa ser útil no dia-a-dia de alguém. A aplicação não pode ser tão simples (como uma mera calculadora), mas também não precisa ser tão complexa (como um editor de imagens estilo Photoshop). Alguns temas válidos para esta aplicação são sistemas para video-locadoras, supermercados, padaria, posto de combustível, agenda para médicos/dentistas, hospital, etc. Embora não seja um requisito, você pode pesquisar/estudar como desenvolver interface gráfica na linguagem Python e utilizar em seu trabalho.
